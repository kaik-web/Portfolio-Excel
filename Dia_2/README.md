# Dia 2 — Relatório de Loja e Busca de Cargo por ID

## Objetivo
Este projeto possui duas planilhas principais:

1. **Relatório de vendas**: Permite analisar os produtos vendidos, identificar vendas estratégicas para investir mais e aquelas mais comuns.  
2. **Busca de cargo por ID**: Permite localizar o nome do funcionário e seu cargo a partir do ID cadastrado.

## Fórmulas utilizadas
- `ÍNDICE + CORRESP`
- `SEERRO`
- `CONT.SE`
- `CONT.SES`
- `SOMASES`
- `PROCX`
