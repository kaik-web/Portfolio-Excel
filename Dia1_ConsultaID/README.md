# Dia 1 — Consulta de Produtos por ID

## Objetivo
Criar uma planilha que permite ao usuário ver o faturamento de cada produto, faturamento total, quantidade de vendas e se a meta de vendas foi atendida:
- Mostrar o Nome
- Mostrar o Preço
- Classificar como Produto Premium ou Comum
- Calcular Faturamento (Quantidade × Preço)
- Tratar erros (ID vazio ou inválido)
